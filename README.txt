README Sudoku Solver

Løser sudokubrett.

1. argument er tekstfil som inneholder størrelse og brett.
2. argument er navn på outputfil.

Kjører best opp til 3x3 brett. Større brett vil ta veldig lang
tid å løse da den bruker brute-force til å finne løsning.

Output.txt er eksempel fra kjøring med 3x3.txt brett.
